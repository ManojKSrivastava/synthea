◆ parseDateRange()

static Range<Long> org.mitre.synthea.helpers.Utilities.parseDateRange ( String  range) throws IllegalArgumentException
static

Parse a range of Synthea timestamps (milliseconds since the epoch) from a String.

The string can be in one of two formats:

"milliseconds start-milliseconds end" example: "1599264000000-1627948800000" or "ISO date start-ISO date end" example: "2020-09-05-2021-08-03"

If using the ISO date format, the time range will be from the start of the day at the beginning of the range until the end of the day for the last day of the range.

Parameters
rangeString containing the range
Returns
A Range with the min and max set to Synthea timestamps, longs containing milliseconds since the epoch
Exceptions
IllegalArgumentExceptionIf the string is not in one of the expected formats
