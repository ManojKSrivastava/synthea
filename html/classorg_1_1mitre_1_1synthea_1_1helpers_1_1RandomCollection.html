Detailed Description

Random collection of objects, with weightings.

Intended to be an equivalent to the ruby Pickup gem. Adapted from https://stackoverflow.com/a/6409791/630384

E  result

Add an object (result) to the collection with a given weight.

If the weight is negative, the object (result) is not added to the collection. Objects are retrieved by weight using the next method.

Parameters
weight- Positive weight of the result.
result- the object to add to the collection.

◆ next()

E org.mitre.synthea.helpers.RandomCollection< E >.next ( RandomNumberGenerator  random)

Select an item from the collection at random by the weight of the items.

Selecting an item from one draw, does not remove the item from the collection for subsequent draws. In other words, an item can be selected repeatedly if the weights are severely imbalanced.

Parameters
randomthe random number generator.
Returns
a random item from the collection weighted by the item weights.
