Detailed Description

This class provides information on the currently Emergency Use Authorized (EUA) vaccines in the United States.

It also provides functionality to select a vaccine based on national usage statistics.

Note that the Pfizer vaccine has received full approval from the FDA, so technically it is no longer under EUA.
