static synchronized Person [] org.mitre.synthea.TestHelper.getGeneratedPeople ( ) throws IOException, ClassNotFoundException
static

This method generates 10 people and then serializes them out into memory as byte arrays.

For tests that need a generated patient, they can call this method to grab a fresh copy of a person which is rehydrated from the byte array to ensure an unmodified copy of the original. This eliminates regeneration of people in the test suite for many of the exporters.

Returns
the people array
Exceptions
IOExceptionwhen there is a problem rehydrating a person
ClassNotFoundExceptionwhen there is a problem rehydrating a person
