Detailed Description

An algorithm that defines the logic for eligibility based on a set of qualifying condition codes.

The codes can either be input as seperated by "xxx|xxx" or can be input in a file with a heading "codes". This can be used to define Social Security eligibility, as described by: https://www.ssa.gov/disability/professionals/bluebook/AdultListings.htm Note that the SSD file (ssd_dsiabilities.csv) list is incomplete, some condtions are not currently simulated in Synthea.
