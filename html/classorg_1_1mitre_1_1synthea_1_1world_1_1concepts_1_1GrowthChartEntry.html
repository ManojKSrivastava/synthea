◆ MAX_PERCENTILE

double org.mitre.synthea.world.concepts.GrowthChartEntry.MAX_PERCENTILE = 0.995
static

The highest percentile to use when attempting to calculate a value.

Tests with the BMI growth charts for females at 240 months show that values greater than 0.99739 cause a term in the equation to go negative causing the code to return NaN. Values slightly less than that can return unrealistic values. This provides a limit to the maximum percentile that will be looked up.
