Detailed Description

Demographics class holds the information from the towns.json and associated county config files.

This data is used to build up a synthetic population matching these real-world statistics. A single instance of Demographics represents a single city or town. The Ages, Gender, Race, Income, and Education properties are maps of frequency information. TODO: add ways to better wrap these maps so they are more accessible and useful. TODO: merge this with Location somehow. they probably don't need to be separate classes

double org.mitre.synthea.world.geography.Demographics.incomeLevel ( int  income)

Simple linear formula just maps federal poverty level to 0.0 and 75,000 to 1.0.

The 75,000 figure was chosen based on https://www.princeton.edu/~deaton/downloads/ deaton_kahneman_high_income_improves_evaluation_August2010.pdf.
