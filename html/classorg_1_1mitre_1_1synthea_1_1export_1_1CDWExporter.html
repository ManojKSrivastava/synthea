Detailed Description

This exporter attempts to export synthetic patient data into comma-separated value (CSV) files that align with the Veteran's Health Administration (VHA) Corporate Data Warehouse (CDW).

https://www.data.va.gov/dataset/corporate-data-warehouse-cdw
