\doxysubsection{Detailed Description}
\mbox{\hyperlink{classorg_1_1mitre_1_1synthea_1_1world_1_1geography_1_1Demographics}{Demographics}} class holds the information from the towns.\+json and associated county config files. 

This data is used to build up a synthetic population matching these real-\/world statistics. A single instance of \mbox{\hyperlink{classorg_1_1mitre_1_1synthea_1_1world_1_1geography_1_1Demographics}{Demographics}} represents a single city or town.
