{\footnotesize\ttfamily Multi\+Table org.\+mitre.\+synthea.\+engine.\+Physiology\+Simulator.\+run (\begin{DoxyParamCaption}\item[{Map$<$ String, Double $>$}]{inputs }\end{DoxyParamCaption}) throws Derivative\+Exception}



Solves the model at each time step for the specified duration using the provided inputs as initial parameters. 

Provides the results as a map of value lists where each key is a model parameter. In addition to the model parameters is a \char`\"{}\+Time\char`\"{} field which provides a list of all simulated time points.

Note that this method will throw a Derivative\+Exception if the model encounters an error while attempting to solve the system.
