\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classorg_1_1mitre_1_1synthea_1_1helpers_1_1SerializableTypeAdapter}{Serializable\+Type\+Adapter}} is a GSON Type\+Adapter used to allow GSON to handle classes that are Serializable, but can\textquotesingle{}t be instrumented by GSON for whatever reason. 

This is primarily used for core Java classes (java.$\ast$.$\ast$) which as of JDK16 no longer allow access via reflection. See\+: \href{https://github.com/google/gson/issues/1216}{\texttt{ https\+://github.\+com/google/gson/issues/1216}}

The internal approach is to write the object to a string using Object\+Output\+Stream, then base64 it. Reading the object reverses the base64 then turns it back into an Object via Object\+Input\+Stream.
