\doxysubsection{Detailed Description}
Task class to generate a properties list of \char`\"{}overridable\char`\"{} fields within the modules, which includes the name of the file, the JSONPath to the field, and the original value. 

By default this is expected to be the set of all distributions, within all transitions, in all modules, but there are 4 configuration options\+:
\begin{DoxyItemize}
\item include\+Fields\+: (defaults to \mbox{[}\char`\"{}distribution\char`\"{}\mbox{]} ) -- numeric fields that match one of the given name will be written to the properties file
\item exclude\+Fields\+: (defaults to null) -- if provided, all numeric fields except those that match the given names will be written to the properties file -- note\+: if both include\+Fields and exclude\+Fields are given, include\+Fields will be ignored
\item include\+Modules\+: (defaults to null) -- if provided, only modules that match the given file names will be processed -- wildcards are allowed, ex \char`\"{}metabolic$\ast$\char`\"{}
\item exclude\+Modules\+: (defaults to null) -- if provided, all modules except those that match the given file names will be processed -- wildcards are allowed, ex \char`\"{}metabolic$\ast$\char`\"{} The format of the properties file is\+: (module file name)\textbackslash{}\+:\textbackslash{}\+:(JSONPath to numeric field within module) = original value Sample Line\+: osteoporosis.\+json\textbackslash{}\+:\textbackslash{}\+:\$\mbox{[}\textquotesingle{}states\textquotesingle{}\mbox{]}\mbox{[}\textquotesingle{}Male\textquotesingle{}\mbox{]}\mbox{[}\textquotesingle{}distributed\+\_\+transition\textquotesingle{}\mbox{]}\mbox{[}1\mbox{]}\mbox{[}\textquotesingle{}distribution\textquotesingle{}\mbox{]} = 0.\+02 
\end{DoxyItemize}
