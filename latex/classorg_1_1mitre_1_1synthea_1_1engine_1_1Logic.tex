\doxysubsection{Detailed Description}
\mbox{\hyperlink{classorg_1_1mitre_1_1synthea_1_1engine_1_1Logic}{Logic}} represents any portion of a generic module that requires a logical expression. 

This class is stateless, and calling \textquotesingle{}test\textquotesingle{} on an instance must not modify state as instances of \mbox{\hyperlink{classorg_1_1mitre_1_1synthea_1_1engine_1_1Logic}{Logic}} within Modules are shared across the population.
