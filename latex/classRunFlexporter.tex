\doxysubsection{Detailed Description}
Entrypoint for the \char`\"{}\+Flexporter\char`\"{} when run as a standalone gradle task. 

The Flexporter is primarily expected to be used to apply transformations to newly generated Synthea patients, but can also be run against existing FHIR bundles. These bundles are assumed to have been generated by Synthea, but depending on the mapping, they don\textquotesingle{}t necessarily have to be.
