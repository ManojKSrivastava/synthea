\doxysubsection{Detailed Description}
This class parses the CSV file that is created by calling the CDC API for COVID-\/19 vaccination information. 

The API can be found at\+: 

\href{https://data.cdc.gov/resource/km4m-vcsb}{\texttt{ https\+://data.\+cdc.\+gov/resource/km4m-\/vcsb}} 

The number of doses given in a day is used to select a date for when an individual in the simulation will get their vaccine. Enumerated\+Distributions are created for each age group, with the days weighted by the number of doses administered on that day.
