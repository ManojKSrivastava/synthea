Synthea\textsuperscript{TM} is a Synthetic Patient Population Simulator. The goal is to output synthetic, realistic (but not real), patient data and associated health records in a variety of formats.

Read our \href{https://github.com/synthetichealth/synthea/wiki}{\texttt{ wiki}} and \href{https://github.com/synthetichealth/synthea/wiki/Frequently-Asked-Questions}{\texttt{ Frequently Asked Questions}} for more information.

Currently, Synthea\textsuperscript{TM} features include\+:
\begin{DoxyItemize}
\item Birth to Death Lifecycle
\item Configuration-\/based statistics and demographics (defaults with Massachusetts Census data)
\item Modular Rule System
\begin{DoxyItemize}
\item Drop in \href{https://github.com/synthetichealth/synthea/wiki/Generic-Module-Framework}{\texttt{ Generic Modules}}
\item Custom Java rules modules for additional capabilities
\end{DoxyItemize}
\item Primary Care Encounters, Emergency Room Encounters, and Symptom-\/\+Driven Encounters
\item Conditions, Allergies, Medications, Vaccinations, Observations/\+Vitals, Labs, Procedures, Care\+Plans
\item Formats
\begin{DoxyItemize}
\item HL7 FHIR (R4, STU3 v3.\+0.\+1, and DSTU2 v1.\+0.\+2)
\item Bulk FHIR in ndjson format (set {\ttfamily exporter.\+fhir.\+bulk\+\_\+data = true} to activate)
\item C-\/\+CDA (set {\ttfamily exporter.\+ccda.\+export = true} to activate)
\item CSV (set {\ttfamily exporter.\+csv.\+export = true} to activate)
\item CPCDS (set {\ttfamily exporter.\+cpcds.\+export = true} to activate)
\end{DoxyItemize}
\item Rendering Rules and Disease Modules with \mbox{\hyperlink{classGraphviz}{Graphviz}}
\end{DoxyItemize}\hypertarget{md_README_autotoc_md10}{}\doxysection{Developer Quick Start}\label{md_README_autotoc_md10}

{\bfseries{System Requirements\+:}} Synthea\textsuperscript{TM} requires Java JDK 11 or newer. We strongly recommend using a Long-\/\+Term Support (LTS) release of Java, 11 or 17, as issues may occur with more recent non-\/\+LTS versions.

To clone the Synthea\textsuperscript{TM} repo, then build and run the test suite\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{git clone https://github.com/synthetichealth/synthea.git}
\DoxyCodeLine{cd synthea}
\DoxyCodeLine{./gradlew build check test}

\end{DoxyCode}
\hypertarget{md_README_autotoc_md12}{}\doxysubsection{Changing the default properties}\label{md_README_autotoc_md12}
The default properties file values can be found at {\ttfamily src/main/resources/synthea.\+properties}. By default, synthea does not generate CCDA, CPCDA, CSV, or Bulk FHIR (ndjson). You\textquotesingle{}ll need to adjust this file to activate these features.
