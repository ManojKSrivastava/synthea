\doxysubsubsection{\texorpdfstring{MAX\_PERCENTILE}{MAX\_PERCENTILE}}
{\footnotesize\ttfamily double org.\+mitre.\+synthea.\+world.\+concepts.\+Growth\+Chart\+Entry.\+MAX\+\_\+\+PERCENTILE = 0.\+995\hspace{0.3cm}{\ttfamily [static]}}



The highest percentile to use when attempting to calculate a value. 

Tests with the \mbox{\hyperlink{classorg_1_1mitre_1_1synthea_1_1world_1_1concepts_1_1BMI}{BMI}} growth charts for females at 240 months show that values greater than 0.\+99739 cause a term in the equation to go negative causing the code to return NaN. Values slightly less than that can return unrealistic values. This provides a limit to the maximum percentile that will be looked up.
