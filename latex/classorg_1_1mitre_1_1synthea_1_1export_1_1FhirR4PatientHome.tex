\doxysubsection{Detailed Description}
Singleton class to manage the instance of the \char`\"{}\+Patient\textquotesingle{}s Home\char`\"{} Location resource. 

FHIR states that when there is a virtual or telehealth encounter, the Location should point to a \char`\"{}kind\char`\"{} resource that represents the patient\textquotesingle{}s home. That means a FHIR based system needs to keep track of the single patient home resource, which is what this class does.
