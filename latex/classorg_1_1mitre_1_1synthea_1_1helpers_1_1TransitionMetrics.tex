\doxysubsection{Detailed Description}
Class to track state and transition metrics from the modules. 

At the end of the simulation this class can print out debugging statistics for each module/state\+:
\begin{DoxyItemize}
\item How many people hit that state
\item What states they transitioned to
\item How long they were in that state (ex, Guard, Delay) 
\end{DoxyItemize}
