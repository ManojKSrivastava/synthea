\doxysubsection{Detailed Description}
Helper class to parse a YAML file into a (hopefully) useable map format. 

This is useful for arbitrary YAML which doesn\textquotesingle{}t try to adhere to any schema. (If it does adhere to a schema, there are other methods to parse YAML to objects which should be preferred) Java doesn\textquotesingle{}t have dynamic typing so this class exposes a \char`\"{}get\char`\"{} method which takes a path string, and walks the map to find the desired item.
