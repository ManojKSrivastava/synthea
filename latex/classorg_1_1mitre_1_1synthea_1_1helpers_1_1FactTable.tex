\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classorg_1_1mitre_1_1synthea_1_1helpers_1_1FactTable}{Fact\+Table}} helper class aids in the export of database-\/style Fact Tables. 

If you have a \char`\"{}table\char`\"{} where each row is a fact or lookup table where a value should be referenced by an ID, you can add these keys and facts to this table and get back the ID.
