\doxysubsection{Detailed Description}
\mbox{\hyperlink{classorg_1_1mitre_1_1synthea_1_1world_1_1concepts_1_1ClinicianSpecialty}{Clinician\+Specialty}} is an enumeration of Clinical Specialties that a Clinician can possess. 

Each of the values (e.\+g. \char`\"{}\+ADDICTION MEDICINE\char`\"{}) corresponds to a column header in the primary care facilities file identified by the Synthea property\+: 

generate.\+providers.\+primarycare.\+default\+\_\+file = \char`\"{}providers/primary\+\_\+care\+\_\+facilities.\+csv\char`\"{} 

The numeric value in each cell identifies how many of clinicians should be generated with each specialty. These specialties can then be used to select an appropriate clinician for any encounter.
