\doxysubsection{Detailed Description}
An algorithm that defines the logic for eligibility based on a set of qualifying condition codes. 

The codes can either be input as seperated by \char`\"{}xxx$\vert$xxx\char`\"{} or can be input in a file with a heading \char`\"{}codes\char`\"{}. This can be used to define Social Security eligibility, as described by\+: \href{https://www.ssa.gov/disability/professionals/bluebook/AdultListings.htm}{\texttt{ https\+://www.\+ssa.\+gov/disability/professionals/bluebook/\+Adult\+Listings.\+htm}} Note that the SSD file (ssd\+\_\+dsiabilities.\+csv) list is incomplete, some condtions are not currently simulated in Synthea.
