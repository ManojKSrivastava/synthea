\doxysubsection{Detailed Description}
\mbox{\hyperlink{classorg_1_1mitre_1_1synthea_1_1engine_1_1Transition}{Transition}} represents all the transition types within the generic module framework. 

This class is stateless, and calling \textquotesingle{}follow\textquotesingle{} on an instance must not modify state as instances of \mbox{\hyperlink{classorg_1_1mitre_1_1synthea_1_1engine_1_1Transition}{Transition}} within States and Modules are shared across the population.
