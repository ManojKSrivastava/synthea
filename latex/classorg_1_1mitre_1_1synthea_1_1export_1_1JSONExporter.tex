\doxysubsection{Detailed Description}
Class that manages the export of a Synthea Person and their related information into a JSON format. 

This is a somewhat \char`\"{}raw\char`\"{} export, as the information is only lightly edited before export. In other words, it\textquotesingle{}s pretty close to an object dump.
