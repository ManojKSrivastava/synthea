\doxysubsection{Detailed Description}
Class exposing various configurable biometric settings, which are set in resources/biometrics.\+yml. 

Unlike the standard \char`\"{}\+Config\char`\"{} and synthea.\+properties, it is unlikely that an end-\/user will want to tweak these often.
