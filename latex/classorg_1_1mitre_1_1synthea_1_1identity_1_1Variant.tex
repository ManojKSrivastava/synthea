\doxysubsection{Detailed Description}
A representation of demographic information for a person. 

The expectation is that it will be a deviation from its associated \mbox{\hyperlink{classorg_1_1mitre_1_1synthea_1_1identity_1_1Seed}{Seed}}, reproducing the variability usually observed in demographic data, such as nicknames, typos, or even placeholder values. \mbox{\hyperlink{classorg_1_1mitre_1_1synthea_1_1identity_1_1Variant}{Variant}} information is only used for supplying information to write into the exported record, so it can contain an incorrect date of birth or non-\/existent city. 

Variants may be sparsely populated. If a value for a field is null, it will pull the value from the associated seed.
